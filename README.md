# PathDuzenleyiciV2
Path Düzenleyicisi (dosya yolu) Path Düzenleyicisi karışık olan sonuçları sizin isteğinize göre düzenleyerek size sunar, bu bir winprefetch den bir sonuç yığını olabilir, systeminformer dan string sonuçları olabilir ve ya Memory çıktılarından URL çıktıları için olabilir.
